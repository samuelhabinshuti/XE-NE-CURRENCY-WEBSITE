                   Currency Exchange System – Full Description

                   This system is a Currency Exchange Website designed to 
                   help users convert money from one country to another. 
                   It allows people to easily exchange currencies such as 
                   converting Rwandan money (RWF) to Chinese Yuan (CNY), 
                   Kenyan Shilling (KES), Ugandan Shilling (UGX), and many others.
                   

                   

                    The main purpose of this system is to provide a simple  
                    and fast way** for users to check exchange rates and 
                    perform currency conversions without difficulty. Users
                    only need to select the currency they have, choose
                    the currency they want, and enter the amount. 
                    The system then calculates and displays the result
                    instantly.

                    


                    This website also provides additional features such as:

                    Viewing exchange rates between different countries
                    Comparing currencies
                    Displaying basic currency information
                    Showing sample historical rates (for learning purposes)

                    The Currency Exchange Website is located in:
                    Western Province, Karongi District, Mubuga Sector**
